Set Wallet Password

The wallet password is different from the portal password. The wallet password is used when signing various transactions that take place in the wallet and is an important security key that allows you to find your wallet if it is lost. We recommend you to set it differently from the portal password.

The password input format is invalid. Password must be a string of 8 or more characters of at least one lowercase, one uppercase, one special character, and a number.
Passwords do not match.
